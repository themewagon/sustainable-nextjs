## Getting Started

Clone from GitHub 
```bash
git clone https://github.com/themewagon/sustainable-nextjs.git
```

Install dependencies:
```bash
npm install
```

Run development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```
